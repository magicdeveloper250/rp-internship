# Instructions
1. Clone this repo into your machine
2. cd rp-internship
3. cd frontend 
4. run "npm i"
5. cd ..
6. cd backend
7. run "npm i"
8. npm start
9. open new terminal
10. cd frontend
11. npm run dev

## Attention

Before cloning or copying the code check if your mongo campus is installed and ready to receive connections.
